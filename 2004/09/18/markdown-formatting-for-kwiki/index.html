Markdown formatting for Kwiki

  • wiki

I just made my first Kwiki formatting plugin, making Markdown formatting available. I'm not sure that it's completely working, but maybe if I get over the lazy threshold, I'll package it up and ship it up to CPAN. For what it's worth, here it is:

package Kwiki::Markdown;
use strict;
use warnings;
use Kwiki::Plugin '-Base';
use mixin 'Kwiki::Installer';

use Markdown;

const class_title => 'Markdown';
const class_id => 'markdown';

sub register {
my $registry = shift;
$registry->add( wafl => markdown => 'Kwiki::Markdown::Wafl' );
}

package Kwiki::Markdown::Wafl;
use base qw( Spoon::Formatter::WaflBlock );

sub to_html {
   Markdown::Markdown($self->block_text);
}

package Kwiki::Markdown;
__DATA__
