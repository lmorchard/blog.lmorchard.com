Ping Pong to Weblogs Dot Com

Simon Willison writes that he'd read my blog more if I pinged weblogs.com more often. I used to, via MovableType, but my new blog doesn't. Enter ping_weblogs_com, a Blosxom plugin to ping weblogs.com. I've just installed it.

Let's see if he notices. :)
