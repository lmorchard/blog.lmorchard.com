# blog.lmorchard.com

Source for [blog.lmorchard.com](https://blog.lmorchard.com) — a personal blog powered by a custom Node.js static site generator.

Posts are written in Markdown with YAML frontmatter. The build system renders them to HTML with indexes, archives, tag pages, and RSS feeds. Client-side enhancements (table of contents, theme switching, search, etc.) use vanilla JS and Web Components.

See [MANUAL.md](MANUAL.md) for details on creating and formatting posts.
