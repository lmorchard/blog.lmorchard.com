.plan

What's this page?

In the ancient past and still today, UNIX users have made .plan files available via the finger protocol. This page is my attempt to provide something with similar intentions, mashed up with Instant Outlining, presented by way of Dan MacTough's OPMLRenderer plugin for WordPress.

If you're using the OPML Editor, you can subscribe to this outline.
